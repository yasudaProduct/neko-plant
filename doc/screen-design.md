# neko-plant 画面設計

**文書の位置づけ:** 本書は、[サービス説明](./service-description-photo-sns.md) §4「ユーザーの動線」で定義した動機（M1〜M7）を実現するために必要な**画面の一覧・各画面の表示項目・画面遷移**を整理した設計ドキュメントです。

---

## 2. 各画面の表示項目

### S01 — フィード（ホーム）

サービスの入口であり、M1（写真を楽しみたい）・M7（初訪問）の起点。

| 領域 | 表示項目 |
|------|---------|
| ヘッダー | サービスロゴ、検索アイコン（→ S04）、投稿ボタン（→ S03）、ユーザーアイコン or サインインリンク |
| キャッチコピー | サービスの一言説明（初訪問者向け、スクロールで隠れてよい） |
| 投稿カード（繰り返し） | 写真（メイン）、投稿者アイコン＋表示名（→ S06）、植物名タグ（→ S05）、猫の名前・種類、いいねボタン＋件数、投稿日時 |
| ページネーション | 「もっと見る」ボタン or 無限スクロール |

### S02 — 投稿詳細

投稿カードをタップした先。M1・M6 で利用。

| 領域 | 表示項目 |
|------|---------|
| 写真 | 拡大表示（複数枚の場合はスワイプ／サムネイル切り替え） |
| 投稿者情報 | アイコン、表示名（→ S06） |
| 植物タグ | 植物名（→ S05） |
| 猫情報 | 猫の名前、種類、写真（あれば） |
| コメント | 投稿者のコメント本文 |
| いいね | いいねボタン＋件数 |
| メタ情報 | 投稿日時 |
| アクション | 自分の投稿の場合: 編集・削除 |

### S03 — 投稿作成

M2（暮らしを共有・残したい）の中核。ステップ形式またはシングルページ。

| ステップ | 入力項目 | 補足 |
|---------|---------|------|
| 1. 写真選択 | 画像ファイル（1 枚以上） | カメラ起動 or ギャラリーから選択。複数枚対応。 |
| 2. 植物の紐付け | 植物タグ（必須） | AI 判定の候補リストから選択、または手動で植物名を検索して指定。 |
| 3. 猫の選択 | 登録済みの猫（任意） | 猫プロフィール未登録の場合は登録への導線（→ S11）を表示。 |
| 4. コメント | テキスト（任意） | 自由記述。 |
| 5. 確認・投稿 | プレビュー＋投稿ボタン | 投稿完了後、S01 またはS02 へ遷移。 |

### S04 — 植物検索

M3（共存を調べたい）・M4（条件で探したい）の入口。

| 領域 | 表示項目 |
|------|---------|
| 検索バー | キーワード入力（植物名・通称） |
| 絞り込み | 猫種フィルタ、タグフィルタ（将来拡張） |
| 並び順 | 共存実績の多い順、名前順、新着順 |
| 結果一覧（繰り返し） | 植物名、学名、サムネイル（投稿写真から）、共存実績サマリー（「〇〇匹の猫と暮らしています」等）（→ S05） |
| 空状態 | 「見つかりませんでした」メッセージ |

### S05 — 植物詳細

M3 のメイン画面。植物ごとの情報と共存実績を集約。

| 領域 | 表示項目 |
|------|---------|
| 植物情報 | 植物名（和名）、学名、科、属、種 |
| 共存実績サマリー | 共存猫数、共存投稿数、表示ランク文言（§5.3 に準拠） |
| 注意喚起 | 投稿がない・少ない場合の注意メッセージ |
| 関連投稿ギャラリー | この植物がタグ付けされた投稿のサムネイル一覧（→ S02） |
| 投稿者一覧（将来） | この植物と暮らしているユーザーのアイコン一覧（→ S06） |

### S06 — ユーザープロフィール

M6（人や投稿を辿りたい）・M5（自分の情報を整えたい）で利用。

| 領域 | 表示項目 |
|------|---------|
| ユーザー情報 | アイコン、表示名、自己紹介文 |
| 統計 | 投稿数、いいねされた数（将来） |
| 投稿ギャラリー | 最新の投稿サムネイル（→ S02）、「すべて見る」（→ S07） |
| 猫プロフィール | 猫の名前、種類、写真（複数匹対応） |
| 植物コレクション | 投稿から自動集計された植物一覧（→ S05） |
| 自分の場合 | 「プロフィール編集」（→ S10）、「設定」（→ S09） |

### S07 — ユーザー投稿一覧

S06 の「すべて見る」先。投稿をグリッドまたはリストで一覧表示。

| 領域 | 表示項目 |
|------|---------|
| ユーザー情報（簡易） | アイコン、表示名 |
| 投稿一覧（繰り返し） | サムネイル、植物タグ、いいね数、投稿日時（→ S02） |
| ページネーション | 「もっと見る」or 無限スクロール |

### S08 — サインイン

M2・M5 で認証が必要な場合の入口。

| 領域 | 表示項目 |
|------|---------|
| サービスロゴ | ロゴ＋一言説明 |
| サインイン方法 | Google OAuth ボタン |
| 補足リンク | 利用規約（→ S14）、プライバシーポリシー（→ S15） |

### S09 — アカウント設定

M5 で利用。

| 領域 | 表示項目・操作 |
|------|--------------|
| メールアドレス | 表示（変更不可 or OAuth 連携情報） |
| 表示名 | 編集フィールド |
| エイリアス ID | 編集フィールド（プロフィール URL に使用） |
| アイコン | 画像アップロード＋プレビュー |
| サインアウト | ボタン |
| 退会 | ボタン＋確認ダイアログ（将来） |

### S10 — プロフィール設定

M5 で利用。

| 領域 | 表示項目・操作 |
|------|--------------|
| 自己紹介文 | テキストエリア |
| 公開設定 | 公開／非公開の切り替え（将来） |
| プレビュー | 変更後の見え方を確認 |

### S11 — 猫プロフィール管理

M2（投稿時に猫を選ぶ前提）・M5 で利用。

| 領域 | 表示項目・操作 |
|------|--------------|
| 猫一覧 | 登録済みの猫カード（名前、種類、写真） |
| 追加 | 「猫を追加」ボタン → 入力フォーム |
| 編集 | 各猫カードの編集ボタン → 入力フォーム |
| 削除 | 各猫カードの削除ボタン＋確認ダイアログ |
| 入力項目 | 猫の名前（必須）、猫種（猫種マスタから選択）、写真（任意） |

### S12〜S16 — 静的・情報系画面

お知らせ、利用規約、プライバシーポリシー、お問い合わせは既存のまま維持。詳細は省略。

### S17・S18 — 管理者画面

| 画面 | 主な操作 |
|------|---------|
| S17 管理者ダッシュボード | 各管理画面へのナビゲーション |
| S18 投稿モデレーション | 投稿一覧の確認、承認・非公開・削除の操作 |

## 3. 画面遷移

### 3.1 動機別の遷移パス

各動機（M1〜M7）がどの画面を辿るかを示す。矢印は主要な遷移方向。

#### M1 — 写真を楽しみたい

```
S01 フィード → S02 投稿詳細 → S05 植物詳細
                             → S06 ユーザープロフィール
S01 フィード → S05 植物詳細（植物タグから直接）
S01 フィード → S06 ユーザープロフィール（投稿者から直接）
```

#### M2 — 暮らしを共有・残したい

```
S08 サインイン → S01 フィード → S03 投稿作成 → S01 フィード or S02 投稿詳細
                                  ↓（猫未登録時）
                                S11 猫プロフィール管理 → S03 投稿作成に戻る
```

#### M3 — 植物と猫の共存を調べたい

```
S04 植物検索 → S05 植物詳細 → S02 投稿詳細 → S06 ユーザープロフィール
S01 フィード → S05 植物詳細（植物タグ経由）
```

#### M4 — 条件を決めて探したい

```
S04 植物検索（絞り込み・並び替え） → S05 植物詳細
                                  → S02 投稿詳細
```

#### M5 — アカウントや猫の情報を整えたい

```
S08 サインイン → S09 アカウント設定
              → S10 プロフィール設定
              → S11 猫プロフィール管理
              → S06 自分のプロフィール（確認）
```

#### M6 — 反応したり、人や投稿を辿りたい

```
S01 フィード → S02 投稿詳細（いいね）
S02 投稿詳細 → S06 ユーザープロフィール → S07 投稿一覧 → S02 投稿詳細
S06 ユーザープロフィール → S05 植物詳細（植物コレクション経由）
```

#### M7 — サービスを初めて知り、把握したい

```
S01 フィード（キャッチコピー＋写真で雰囲気を掴む）
  → M1 の動線（閲覧を続ける）
  → S04 植物検索（調べたい場合）
  → S08 サインイン（参加したい場合）
```

### 3.2 グローバルナビゲーション

| サインイン（未ログイン） | S08 サインイン |

| 画面 | 状態 | 対応内容 |

| S09 アカウント設定 | 既存 | 変更なし。 |
| S10 プロフィール設定 | 既存 | 変更なし。 |
| S11 猫プロフィール管理 | **新規** | `/settings/pets` を新設。猫の CRUD。 |
